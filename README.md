## 📁 Managing Your Files

The main screen shows your device once it connects. You can select your desired task from the menu icons.

### Backing Up Data
Select the Backup tab to secure your device contents. Click the Start Backup button to begin. The program copies your contacts, messages, and settings to a folder on your computer. You can choose the storage location by clicking the settings icon.

### Cleaning Photos
Select the Gallery tab to see your photos. The software displays your images in a grid format. Click an image to select it. Click the Delete button to remove files from your phone. You can sort images by date or file size to find large files easily.

### Navigating Storage
Select the File Browser tab to see the hidden file structure of your iPhone. This feature gives you access to DCIM folders where your camera saves photos. You can drag and drop files from these folders to your Windows desktop to save them.

## ❓ Frequently Asked Questions

**Does this software send my data to a server?**
No. This program works offline. It never connects to the internet or any cloud service. Your data stays on your local drive.

**Can I restore a backup with this tool?**
Yes. Use the Restore tab to pick a previous backup file from your computer and move it back to your iPhone.

**Why does my phone ask for a passcode?**
The iPhone requires a passcode to ensure only you can access your data. Keep your phone unlocked while using the software.

**What happens if I disconnect the cable during a process?**
The software stops the current task. Always wait for the progress bar to show complete before unplugging the cable to prevent data corruption.

**Do I need iTunes installed?**
No. This software handles the connection process without relying on external drivers or programs like iTunes.
